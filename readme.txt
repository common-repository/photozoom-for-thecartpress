== Description ==

Allows to display featured images with Zoom in your eCommerces made using TheCartPress, eCommerce plugin for WordPress.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload TheCartpress PhotoZoom to '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
